# AWS SAA-C03 学習ノート

AWS Solutions Architect Associate（SAA-C03）の学習サイトです。

## 構成

全10章で、AWSの主要サービスを体系的に学びます。

| 章 | テーマ |
|----|--------|
| 第1章 | グローバルインフラ・ネットワーク基礎 |
| 第2章 | IAM・ガバナンス・セキュリティ |
| 第3章 | コンピューティング |
| 第4章 | ストレージサービス |
| 第5章 | データベースサービス |
| 第6章 | ネットワーキング・コンテンツ配信 |
| 第7章 | アプリケーションサービス |
| 第8章 | 分析・データ転送 |
| 第9章 | 運用支援サービス |
| 第10章 | 総合演習 |

## 各章の構成

各章は以下のセクションで構成されています。

- **座学** — 講義形式のナレーションで概念を学ぶ
- **練習問題** — 試験形式の問題を解く
- **問題解説** — 正解と詳しい解説
- **ハンズオン** — 実際にAWSコンソールで操作する

## 学習の進め方

1. 座学を読んで概念を理解する
2. ハンズオンで手を動かす
3. 練習問題を解く（解説を見る前に自力で）
4. 問題解説で理解を深める
